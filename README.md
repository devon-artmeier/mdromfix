# Sega Genesis/Mega Drive ROM Fixer

This is a tool to pad and calculate checksum for Sega Genesis/Mega Drive ROMs.

## Usage

    mdromfix <-q> <-d> <-p [pad value]> [rom file]
    
        <-q>             - Quiet mode
        <-d>             - Don't pad
        <-p [pad value]> - Pad byte value
        [rom file]       - ROM file
